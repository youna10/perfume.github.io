SmartPerfume

한가지 향인 아닌 여러가지 향을 앱을 이용하여
상황, 기분에 따른 향

  • Type

    Application

  • Tool

    android studio, audio, PPT

  • 참여도

    79%(Team Project)

방향제와 앱을 연동하여 스마트폰으로 이용할 수 있는 편리

여러가지 향 선택으로 선택의 제한을 해소

무드등을 접목시켜 디자인 요소를 추가하여 감성을 유도

음성인식을 통해 기분에 따른 향 분사

순서도

SMARTPERFUEME의 작동과정의 순서도를 기반으로 프로젝트개발을 진행했습니다.

회로도

캐드를 이용하여 아두이노와 모듈간의 연결을 그림을 나타내
연결에 어려움이 있을 때 그림을 이용하여 쉽게 파악했습니다.

외형 설계도

스케치업을 통해 외형을 디자인했으며 3D프린트를 사용하여 제작했습니다.

APP 사용가이드

android studio를 이용하여 앱 개발했으며 상세화면에대한 가이드입니다.

블루투스 모듈을 이용하여 스마트폰과 SMART PERFUME 연결

사용자가 원하는 시간예약에 따라 방향제 제어

사용자가 원하는 향을 선택하고 OK버튼을 누르면 향 분사

무드등기능으로 사용자가 원하는 색상을 선택시 원하는 색 출력

음성인식을 통해 사용자의 기분에 따른 추천향을 분사
